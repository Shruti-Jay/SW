Handles command line arguments for the program.

Parses the input arguments, if the file is found, returns the file name. If the file is not found, prompts the election official for another file name. It does this until it is given a valid file name and returns it.

Parameters
argcThe number of command line arguments.
argvAn array containing the command line arguments.
Returns
A string containing the file name of the election to be ran.
