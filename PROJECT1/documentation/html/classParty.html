◆ firstAllocation()

int Party::firstAllocation ( int  quota)

Number of seats won during the first allocation.

Calculate the number of seats won during the first allocation via a quota. To determine the first allocation of seats we use this formula:

  • numSeatWon: the quotient of numVotes/quota.
  • numRemainingVotes: the remainder of numVotes/quota. The first allocation of seats can never be greater than the number of candidates in a party.
Parameters
quotaThe threshold to guarantee a seat.
Returns
The number of seats won during the first allocation.
