std::vector< T * > Election::tieHandler ( std::vector< T * >  unsorted)
inline

Handles a Tie.

Takes in a list of Parties or Candidates and returns a list sorted by the number of votes. If there is a tie, the tie is broken randomly, as the list is shuffled 1000 times, then sorted with a stable sort.

Template Parameters
TGeneric type to be sorted, is either Party or Candidate.
Parameters

Precondition
The generic type T must have operator> defined.
Returns
The sorted list.
