Detailed Description

The entry point for the voting system.

Holds the main program loop of the voting sytem. Starts by either recieving the file name from the command line or if not, prompting the user for the file name. Then enters the main loop which repeats whenever there is an error, if there is an error, the sytem re-prompts for a file name. Inside of the main loop, the program will run the election, then output the results.
