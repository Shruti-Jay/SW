Detailed Description

The Closed Party List Election.

The election type for closed party list (CPL). Input files for CPL will have the standard # ballots and seats, but will also have # parties. Parties and Candidiates are listed as a list of parties with affiliated candidates in a list. When the election is run, since parties are voted for, the final allocation of seats is based on party popularity as well as candidate order listed in the input file.
